Présentation

Les Observables font partis de la librairie RXJS.

Ce sont des fluxs asynchrones, qui peuvent envoyer plusieurs données.

Pour récupérer les données, il faut souscrire au retour de l'Observable.

Les observables peuvent envoyer 3 types de réponses :

  • Next : les données attendues
  • Error : en cas de soucis, elles arrêtent l'observable
  • Complete : indique que le flux se coupe, et qu'il n'y aura plus de données.

Attention : il faut se désabonner des fluxs ouverts ( ceux qui n'entrent pas dans le complete)
