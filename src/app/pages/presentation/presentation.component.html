Interpolation

On peut interpoler du JS ( du TS ) grâce aux balises moustaches

  • 1 + 1 = {{ 1 + 1 }}
  • Donnée du ts : {{ data }}
  • User:
    • Nom: {{user.nom}}
    • Age : {{user.age}}

Property Binding

Tous les attributs HTML existants peuvent être renseigner dynamiquement, en interprétant du JS. Pour ça, il faut mettre le nom de l'attribut entre []

Exemple : [class]='nom_de_variable'
